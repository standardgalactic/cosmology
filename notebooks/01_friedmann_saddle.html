RSVP Notebook 1 — The Friedmann Saddle

Visualising the unstable rest point SM = (4/3, 2/3) in self-similar coordinates

Physics

The critical Friedmann spacetime SM is an unstable saddle rest point of the STV-ODE. In log-time τ = ln t, the linearised flow at SM has eigenvalues:

λ_A^n = 2n/3 > 0
λ_B^1 = −1 < 0 (gauge)
λ_B^2 = −1/3 < 0 (physical)

Particles released near SM roll away along the unstable manifold toward M (Minkowski attractor), exhibiting accelerated expansion.
